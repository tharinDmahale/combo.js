# combo.js
Full stack express.js based web development framework.

---

## How to run

1. At a desired directory enter `npm install -g create-combo-app`.
2. Then run `create-combo-app <project-name>` with a desired project name.
3. At the root level of the project give the command `npm run app`.
4. Then, a new tab will open in your browser with the URL [http://localhost:3001/](http://localhost:3001/).
5. If not, open your browser and type the URL [http://localhost:3001/](http://localhost:3001/).
